# Project Statement

## Problem Statement
Users often need a simple offline tool to manage their day‑to‑day tasks without relying on large apps or internet access.

## Scope
This project provides task creation, deletion, completion tracking, and persistent storage using JSON.

## Target Users
- Students  
- Professionals  
- Anyone needing a lightweight task manager  

## High-Level Features
- Add task with due date  
- List tasks sorted by date  
- Mark tasks complete  
- Delete tasks  
- JSON-based persistent storage
