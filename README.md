# Console To‑Do App

## Overview
A simple Python console application that helps users manage tasks using JSON-based persistence.

## Features
- Add tasks  
- View tasks (auto‑sorted by date)  
- Mark tasks as complete  
- Delete tasks  
- JSON storage  
- Error & input validation  

## Technologies Used
- Python  
- JSON file storage  

## How to Run
```bash
python main.py
```
